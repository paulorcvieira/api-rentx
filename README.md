# RentX

**Clone this repository**

```bash
$ git clone -e https://github.com/paulorcvieira/api-rentx
```

**Rename files**
```bash
$ mv ormconfig.json.exemple ormconfig.json
$ mv .env.exemple .env
```

**Access the root folder**

```bash
$ npm install
$ npm run up
```

**if first running**
```bash
$ docker exec -it api_rentx_container /bin/bash
$ npm run seed:admin
$ exit
```
